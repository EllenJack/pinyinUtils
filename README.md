# pinyin #

----------

***pinyin*** is java library which could translate Chinese character into Chinese PinYin.

## maven build ##
```

## usage

```java
Pinyin py = new Pinyin();
py.translate("汉");
py.translate("わたしわ数据, and my English name is Tom. ^_^。下面是一段多音分词歧义测试，冬天：能穿多少穿多少；夏天：能穿多少穿多少。");
```
